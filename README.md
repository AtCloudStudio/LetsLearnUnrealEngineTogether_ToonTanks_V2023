![""](Images/Cover.png)

## 【项目学习完成】

本项目学习的是GameDev.TV在Udemy发布的教程。
主要内容是一个Top-down类型的射击类小游戏。
个人觉得非常适合虚幻引擎的C++编程入门，因此在直播时使用了这个教程的内容。
如果是从Unity转到UE的话，学习上面提到的教程则会更有优势。
